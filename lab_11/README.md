# ЛР11

## Задание

Написать программу для работы с записями одинаковой структуры в текстовой файле.

Меню:

* Выбор файла
* Создание нового набора записей
* Добавление записей
* Вывод всех записей
* Поиск по одногому полю
* Поиск по двум полям

Записи хранить в одной строке\
В записи 3-4 поля, разделенные пробелами.
